一种动态样式语言，css预处理语言的一种，包含
变量，继承，运算，函数等。
编译工具：
1，koala不要勾选设置里面的使用系统的less编译器
	否则出现乱码报错
2，node.js库
3，浏览器端

注释：
/*被编译到css文件*/
//不被编译到css文件

变量：@开头
声明：@value_name:100px;
使用：width：@value_name;

混合：重用样式
.box1{}
.box2{.box1;再加自己独有样式}
对于重用的样式，如果重用样式的值需要不同的话？
可以通过带参数解决：
.box(参数一，参数二){}
让有参数的可以有默认值，在参数中给值
.box(参数一:300px){}
使用时如果重要默认值：.box();

匹配模式：
.trangle(top,参数1，参数2){}
.trangle(top,参数1，参数2){}
.trangle(@_,参数1，参数2){}不管匹配那个模式，这段都被执行。
使用：.sanjiao{.trangle(top)}

运算:
运算时，表达式中只要有一个带了px就可以，需要的话。

嵌套：嵌套层数尽量少。
.list{
	li{
	
	}
	a{
	
	}
}

&代表上一层选择器
&:hover{
}

@arguments变量
包含了所有传递进来的参数。
